Cyber-Typer ⚡
A minimalist typing speed test built for the Macondo event. This project focuses on a clean "Cyber-Protocol" aesthetic and helps users practice typing cybersecurity and programming terminology.

🚀 Live Demo
Click here to play Cyber-Typer

✨ Key Features
Themed Dictionary: Practice with words like firewall, javascript, and encryption.

Live Stats: Tracks your WPM and Accuracy as you type.

Minimalist Design: Dark olive "Cyber" theme inspired by Monkeytype.

Test History: View your recent scores at a glance.

🛠️ Built With
HTML5

CSS3 (Flexbox & Custom Variables)

Vanilla JavaScript

⌨️ How to Play
Start typing the words on the screen.

Use Tab + Enter to quickly reset the test.

Your stats will update automatically at the end of the 30-second timer.
